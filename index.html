What kinds of programming languages are there?

Now… What types of programming languages are there. First of all there are two different programming language bases… there is imperative languages, where you tell the computer how to do something, and then there declarative programming languages, where you tell the code to do something. Since declarative programming is used today almost all the time, there are two types in there. There are object oriented programming languages, which use classes like Python, Java, C#, ect., and then there are procedural programming languages which is like Javascript, HTML5 (which is what this app/webpage is using), and Lua!
